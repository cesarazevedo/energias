Aula 1.1 - Introdução à Lógica de Programação

Slide 1
O que é Lógica?

🧠 Lógica é pensar de forma organizada!

Imagine que você quer fazer um sanduíche. Você não pode colocar a maionese antes do pão, certo? Isso é lógica!

  • Lógica: Forma correta de pensar e resolver problemas
  • Sequência: Ordem certa das ações
  • Raciocínio: Como chegamos na solução

🍞 Exemplo do Sanduíche:

1. Pegar o pão
2. Cortar o pão
3. Passar maionese
4. Colocar recheio
5. Fechar o sanduíche

Slide 2
Lógica no Computador

💻 O computador é como uma criança obediente

Ele faz EXATAMENTE o que mandamos, na ordem que mandamos!

  • Computador não "pensa" sozinho
  • Precisa de instruções claras e detalhadas
  • Não entende "mais ou menos" - é tudo ou nada

🤖 Exemplo:

Se você disser "esquente a água", o computador não sabe quanto esquentar. Precisa ser específico: "esquente a água até 100°C"

Slide 3
Problemas do Dia a Dia

🎯 Resolvendo problemas passo a passo

Todo problema pode ser quebrado em pequenos pedaços!

  • Entender o problema: O que queremos resolver?
  • Dividir em passos: Quais ações precisamos fazer?
  • Organizar a ordem: Qual a sequência certa?
  • Testar: Funciona? Se não, onde erramos?

🏠 Problema: Como chegar em casa?

1. Sair da escola
2. Caminhar até o ponto de ônibus
3. Pegar o ônibus 123
4. Descer na rua X
5. Caminhar 2 quadras
6. Chegou!

Slide 4
Exercício Prático

🎮 Vamos praticar!

Pense no seu problema favorito e vamos resolver juntos:

🕹️ Problema: Como jogar um jogo no celular?

1. Pegar o celular
2. Desbloquear a tela
3. Procurar o ícone do jogo
4. Tocar no ícone
5. Esperar carregar
6. Começar a jogar!

Dica: Sempre teste sua lógica! Se alguém seguir seus passos, vai conseguir fazer igual?

🎯 Exercícios da Aula

Questão 1: Descreva os passos para escovar os dentes (mínimo 5 passos)

Questão 2: Por que é importante ter uma sequência lógica para resolver problemas?

Aula 1.2 - Algoritmos Básicos

Slide 1
O que é um Algoritmo?

📋 Algoritmo é uma receita!

Assim como uma receita de bolo, um algoritmo é uma lista de instruções para resolver um problema.

  • Algoritmo: Sequência de passos para resolver um problema
  • Finito: Tem começo, meio e fim
  • Claro: Cada passo deve ser bem explicado
  • Eficiente: Resolve o problema da melhor forma

🍰 Receita de Bolo (é um algoritmo!):

1. Misture 3 ovos
2. Adicione 2 xícaras de farinha
3. Mexa bem
4. Asse por 30 minutos
5. Bolo pronto!

Slide 2
Características dos Algoritmos

✅ Todo bom algoritmo tem:

  • Entrada: O que precisamos para começar (ingredientes)
  • Processamento: O que fazemos (misturar, assar)
  • Saída: O resultado final (bolo pronto)
  • Precisão: Cada passo deve ser exato
  • Finitude: Deve terminar em algum momento

📱 Algoritmo: Ligar para alguém

Entrada: Telefone e número
Processamento: Discar o número
Saída: Ligação realizada

Slide 3
Formas de Representar Algoritmos

📝 3 maneiras de escrever algoritmos:

  • Descrição Narrativa: Como uma história
  • Fluxograma: Desenho com símbolos
  • Pseudocódigo: Parecido com programação

🚿 Exemplo: Tomar banho

Narrativa: "Primeiro abra o chuveiro, depois ajuste a temperatura, em seguida se ensaboe e por fim se enxágue."

Pseudocódigo:
INÍCIO
  Abrir chuveiro
  Ajustar temperatura
  Se ensaboar
  Se enxaguar
FIM

Slide 4
Primeiro Algoritmo

🏁 Vamos criar nosso primeiro algoritmo!

Problema: Calcular a média de duas notas

ALGORITMO CalcularMedia INÍCIO Escreva "Digite a primeira nota:" Leia nota1 Escreva "Digite a segunda nota:" Leia nota2 media = (nota1 + nota2) / 2 Escreva "A média é:", media FIM

Explicação:

  • Pedimos duas notas ao usuário
  • Somamos as notas e dividimos por 2
  • Mostramos o resultado

Aula 5.1 - Introdução à Linguagem C

Slide 1
História da Linguagem C

📚 Como nasceu a linguagem C?

A linguagem C foi criada nos anos 1970 por Dennis Ritchie. É como o "avô" de muitas linguagens modernas!

  • 1972: Criada por Dennis Ritchie na Bell Labs
  • Objetivo: Programar o sistema operacional UNIX
  • Característica: Simples, mas muito poderosa
  • Influência: Base para C++, Java, C# e outras

🌳 Árvore Genealógica das Linguagens:

C (1972) → C++ (1983) → Java (1995) → C# (2000)

É como uma família! C é o bisavô de muitas linguagens atuais.

Slide 2
Por que aprender C?

🎯 Vantagens da linguagem C:

  • Eficiência: Programas rápidos e leves
  • Portabilidade: Funciona em qualquer computador
  • Fundação: Base para entender outras linguagens
  • Controle: Você controla cada detalhe do programa
  • Amplamente usada: Sistemas operacionais, embarcados

🚗 Analogia com Carros:

C é como aprender a dirigir um carro manual. É mais difícil no início, mas você entende melhor como tudo funciona. Depois, dirigir automático (outras linguagens) fica fácil!

Slide 3
Estrutura de um Programa em C

🏗️ Como é um programa em C:

#include <stdio.h> // Biblioteca para entrada/saída int main() { // Função principal // Aqui vai o código printf("Olá Mundo!"); return 0; // Programa terminou com sucesso }

Explicação das partes:

  • #include: Importa funções prontas (como printf)
  • main(): Onde o programa começa a executar
  • {}: Delimitam blocos de código
  • return 0: Indica que tudo deu certo
Slide 4
Compilação vs Interpretação

⚙️ Como o computador entende C?

C é uma linguagem compilada. Isso significa que precisamos "traduzir" nosso código antes de executar.

📖 Processo de Tradução:

1. Código fonte: Você escreve em C (main.c)
2. Compilador: Traduz para linguagem de máquina
3. Executável: Arquivo que pode rodar (.exe)
4. Execução: Programa roda no computador

Vantagem: Programa fica mais rápido!
Desvantagem: Precisa compilar sempre que mudar algo.
